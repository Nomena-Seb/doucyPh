# Fonts

Use these exact files for the product typography:

- /public/fonts/futura-medium/Futura-Medium.woff2
- /public/fonts/cream-cake/Cream-Cake.woff2
- /public/fonts/chanson-romantique/Chanson-Romantique.woff2

These files are loaded by @font-face in src/index.css.
If a file is missing or empty, the browser will fallback to a default font.
